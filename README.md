## Laravel Bootstrap 5, Tailwind, Livewire and Jetstream

## Preinstalled with

- Bootstrap 5.0.2
- FontAwesome 5.15
- Laravel 8.49.1
- Tailwind CSS 2
- Laravel Livewire
- Jetstream

## Pre-install

```
composer install
npm install && npm run dev
php artisan migrate
php artisan db:seed
php artisan storage:link
php artisan key:generate
```
